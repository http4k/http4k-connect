# Cognito

The Cognito connector provides the following Actions:

- AdminCreateUser
- AdminDeleteUser
- AdminDisableUser
- AdminEnableUser
- AdminGetUser
- AdminResetUserPassword
- AdminSetUserPassword
- AssociateSoftwareToken
- CognitoAction
- CreateUserPool
- CreateUserPoolClient
- DeleteUserPool
- DeleteUserPoolClient
- InitiateAuth
- RespondToAuthChallenge
- VerifySoftwareToken
